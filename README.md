# 📊 Dashboard Interativo em Power BI

Este projeto consiste no desenvolvimento de um **dashboard interativo em Power BI** para análise de indicadores de desempenho, consolidando dados em uma visualização única e dinâmica.

## 📝 Descrição do Projeto
- Estruturação de **visualizações dinâmicas** (gráficos, tabelas e KPIs) que apoiam a tomada de decisão estratégica por gestores de **RH** e **Finanças**.
- Aplicação de filtros interativos para permitir análises detalhadas e comparativas.
- Demonstração da versatilidade do Power BI em análises de **[vendas de jogos](ca://s?q=Análise_de_vendas_de_jogos)**.

## 📊 Estrutura dos Dashboards
Foram desenvolvidos **4 dashboards principais**:
1. **[Vendas por gênero](ca://s?q=Dashboard_vendas_por_gênero)**  
   - Análise das vendas segmentadas por categorias de jogos.
2. **[Vendas por jogo](ca://s?q=Dashboard_vendas_por_jogo)**  
   - Visualização detalhada do desempenho individual de cada título.
3. **[Vendas por plataforma](ca://s?q=Dashboard_vendas_por_plataforma)**  
   - Comparação entre diferentes consoles e sistemas.
4. **[Vendas por publisher](ca://s?q=Dashboard_vendas_por_publisher)**  
   - Identificação dos publishers com maior participação nas vendas.

## 🛠️ Tecnologias Utilizadas
- **Power BI** para construção dos dashboards.
- **KPIs interativos** para acompanhamento de metas.
- **Filtros dinâmicos** para segmentação dos dados.

## 🚀 Funcionalidades
- Visualizações claras e interpretáveis.
- Dashboards interativos com filtros por gênero, jogo, plataforma e publisher.
- KPIs estratégicos para acompanhamento de desempenho.
- Suporte à análise comparativa entre diferentes dimensões de vendas.

## 📈 Resultados
- Consolidação de dados em uma única ferramenta visual.
- Apoio direto à tomada de decisão estratégica.
- Facilidade na identificação de padrões e insights sobre o mercado de jogos.
